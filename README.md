# HIT and BLOW

## 概要
ヒットアンドブローで遊べるwebアプリです。

## 使用技術
- HTML / CSS
- Python / Flask
- SQLite3

## 開発環境
- Python 3.13.3
- Flask 3.1.1
- Gitによるバージョン管理
- 開発にはVSCodeを使用

## 起動方法(ローカル)
1.必要なパッケージをインストール
~~bash
pip install flask
~~
2.実行
~~bash
python app.py
~~
